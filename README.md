# infra-roadmap
План состоит в том, чтоб за 3 года стать инженером уровня Senior Platform Engineer / Infrastructure Architect, который может:

- проектировать инфраструктуру предприятия;
- писать backend на Python;
- создавать внутренние сервисы;
- проектировать Kubernetes-платформы;
- строить CI/CD;
- автоматизировать всё через Ansible/Terraform;
- использовать ИИ как инструмент разработки;
- руководить технической командой;
- иметь портфолио уровня сильного Senior.
